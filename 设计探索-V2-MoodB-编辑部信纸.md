## 1. 产品 DNA（30 秒读完）

**时间银行**是一个 iOS App，把"时间"当成银行账户来使用。

它有两层叠加：

- **消耗层**：自动算出你和父母/孩子/伴侣"还能共度多少小时"、运动/创造/自由"还剩多少时间"。这些数字会一直减少，但不是焦虑倒计时——是**温柔提醒"现在很值钱"**。
- **存储层**：用户可以手动把"有意义的瞬间"（带图文）存进对应的时间账户。这些被存入的内容**永不被消耗、永不被时间带走**。

每个时间账户卡片同时显示两个数字：「还能共度 552h」+「已存入 56h」。

哲学源头：《四千周》(Oliver Burkeman) + 银行账户隐喻。**核心不是"剩余倒计时焦虑"，而是"被感受到的时间不会被带走"**。

调性：**克制 · 温暖 · 不焦虑 · 让人每天想点开看一下**。

## 4. Mood「编辑部信纸」（本次唯一 mood · 严格执行）

> 灵感参考：Headspace / Calm / 纽约时报 The Daily / Aesop 官网 / 月度文学杂志专栏
>
> 关键词：杂志感 editorial / 深底反白 / 强对比 / 大字 hero / 信封感

### 色板（必须用这组，不要替换）

| Token | 色值 | 用途 |
|---|---|---|
| 背景 | `#1F2D26` | 整体背景（深墨绿，**不是黑**）|
| 卡片 | `#2A3933` 或 `rgba(255,255,255,0.04)` | 卡片表面，半透明叠加 |
| 文字主 | `#F4EFE3` | 标题、大数字（米白）|
| 文字次 | `#BFB8A6` | 副文案、元信息 |
| 文字三 | `#877F6E` | 占位文字、tertiary label |
| Accent 金黄 | `#D4A35F` | **数字 hero 用 + 标志性元素**（每屏 1-2 处）|
| 分隔线 | `rgba(244,239,227,0.12)` | 半透明米白 hairline |

> **强对比，但不刺眼**。绿色基调而不是冷蓝，保持温暖。Accent 金黄是温柔的金，不是刺眼的黄。

### 字体

| 用途 | 推荐字体 |
|---|---|
| 标题（hero）| **Playfair Display Bold / 思源宋体 Bold / Cormorant Garamond Bold** |
| 正文 | Helvetica Neue / 思源黑体 Light / Inter |
| 数字（关键 hero）| **大且粗的 serif，类似杂志章节编号** —— 推荐 Playfair Display Black 或 IBM Plex Serif Bold |

### 卡片视觉

- 卡片像信封展开：底色微微比背景亮一点（`#2A3933`），加 1px 半透明米白边框
- 可加非常轻微的 vintage 纹理（noise 5-8% opacity），让背景不死板
- 大字标题 + 细分隔线 + 大段留白分区
- **时间数字可以大到 80-120pt 当 hero 用**（这是 Mood B 最大特征）
- 圆角小（4-8pt）或干脆方角，强化"信纸/书页"感

### 排版细节

- 标题字号反差极大（hero 数字 80pt vs section header 14pt）
- 行间距 1.4-1.6
- 标题前可加一个细小的章节符号（• / § / —）
- 中文标点用全角

### 整体感

像在读一本月度杂志的某个专栏，文字有重量感，每个数字都像编辑亲笔写下的章节标题。**信封感**——内容像被精心包裹好的礼物。

> ⚠️ 虽然全局是深底，但要保持温暖（绿色基调而不是冷蓝），不能有焦虑感。**这不是 Dark Mode，是设计选择**。

## 5. 硬约束（必须守）

1. **设备**：iPhone 16 Pro 竖屏，**渲染尺寸 393×852 pt**（@3x = 1179×2556 px 静态图）
2. **Light Mode only**（V1 不做 Dark Mode · Mood B 的"深底"是设计选择不是 Dark Mode）
3. **双数字时间账户卡片**：消耗层 + 存储层两个数字必须同时显示，缺一不可——这是产品识别物
4. **lifespan 双层卡**：上下两半（时间余额 + 已存入）必须同时存在
5. **3 Tab + 中央 FAB**：底部 nav 不能改成抽屉、汉堡、或单 Tab
6. **文案逐字按 §3 给的**：不要改"陪父母"为"父母"、不要改"已存入"为"已记录"、不要改"还能共度"为"剩余"
7. **不要任何徽章、勋章、连续打卡日历、进度条、KPI 数字**
8. **不要"今天还差 2 小时完成目标"这类提示**——这违反产品哲学
