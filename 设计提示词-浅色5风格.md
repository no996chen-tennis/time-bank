# 时间银行 · 浅色方向 5 份提示词（V1.1 · 2026-04-22）

> 基于 Adam 对「设计提示词-5风格.md」的反馈：
> - **喜欢 Headspace（风格 2）和 Zen（风格 4）两个方向**
> - **整体希望浅色**（砍掉 Claude 风/Apple 风/冥想风三个偏深色的候选）
> - **参考 Dribbble 再找 3 个不同方向**
>
> 本文 5 份提示词全部锁定浅色主色板。

---

## ⚠️ 关于"从 Dribbble 找作品"的诚实说明

Dribbble 的页面是 JS 渲染的单页应用，我的 WebFetch 工具拿不到具体图像和排版描述（只返回空页面）。所以**我不能装作"我看过具体 XX 号作品觉得很适合"**。

我改用更靠谱的方式：

1. **给你已验证存在的 Dribbble tag 页面 URL**——这些页面聚合了同类作品几百张，你点进去 5 分钟就能刷到喜欢的
2. **给你具体的设计师/工作室名字**——在 Dribbble 搜索框直接搜他们，作品质量有保障
3. **给你现实已发布的 App 名字**——作为"成品"参照，比 Dribbble 稿子更可靠（稿子是概念，App 是真的工程实现过）
4. **每份提示词明确写了风格定位**——即使 Adam 懒得浏览，Claude Design 也能靠提示词生成合理结果

## 方向总览

| # | 方向 | 一句话调性 | 和其他风格的差异 |
|---|------|-----------|----------------|
| **1** | Warm Illustration（Headspace 浅色版，锁定） | 温暖治愈 · 插画承载 | 有插画、有手绘感、圆角大 |

| **5** | Editorial Minimal（杂志编辑 · 新） | 大号 Serif · 字体驱动 | Typography-first、承载重量 |

## 视觉语言

**色彩（全浅色，V1 锁定一套）**
- 主背景：#FBF4EC（奶咖色）
- 卡片背景：#FFFFFF
- 次级背景：#F5ECE0（较深一档的米黄）
- 主文字：#2D2520（深可可，不是纯黑）
- 次文字：#8B7968
- 时间账户色（低饱和珊瑚系）：
  - 陪父母 · 珊瑚橘 #E89A7C
  - 陪孩子 · 金黄 #E8C47C
  - 陪伴侣 · 夕阳紫 #C29DD4
  - 运动 · 薄荷绿 #8EC7A8
  - 创造 · 晨雾蓝 #A8C4D9
- 所有色彩饱和度 ≤ 60%，温暖不刺眼

**字体**
- 中文：阿里普惠 Medium 或 OPPO Sans Medium
- 英文/数字：DM Sans、Poppins、Fraunces（如需 Serif）
- **Medium 500 是默认字重**（不要 Light 不要 Bold）
- 大数字可以 Semibold，但必须是圆润字形

**插画与图形**
- 每个时间账户 icon 是一张小插画：陪父母 = 长凳上两个剪影 / 陪孩子 = 小手牵大手 / 陪伴侣 = 面对面 / 运动 = 清晨跑道
- 允许情绪化符号：温柔的眼睛、伸开的手掌、抱着的人、光晕
- 圆角全部 20pt+，卡片 24pt，按钮 100pt（胶囊）
- 允许暖色渐变（橙→粉→黄），禁止冷色渐变
- 按钮可有柔和暖色阴影（offset 0 4 12 #E89A7C 15%）
- 插画风格：细线条 + 大色块组合

**布局**
- 中等密度，不必极端留白
- 主色块大，可铺满卡片背景
- 文字可居中
- Tab Bar 明显可见，有插画式 icon

**动效**
- Spring 弹性明显，damping 0.7-0.8
- 按钮按下有"呼吸"感（scale 0.95 带回弹）
- 存入成功：心形光点 + 放射微光 + 轻微震动
- Onboarding 页间用 slide + scale

## 必设计的 6 屏（iPhone 16 Pro 393×852）

1. **Welcome 启动页** — 一幅温暖插画（两人坐在窗前）+ 标题 + 按钮
2. **Onboarding Step 2** — 4 个 chip（带小插画 icon）"父母健在 / 有孩子 / 有伴侣 / 独处也是一种时间"
3. **主页** — 顶部总账户 + 双数字时间账户卡片（左侧插画 + 右侧数字）
4. **时间账户详情（陪父母）** — 顶部大插画（两位老人剪影）+ 大数字 "552h" + 参数卡 + 时间线
5. **新增时刻表单** — 字段带可爱 placeholder 插画
6. **时刻详情** — 媒体轮播 + 温暖 chip + 笔记文字

## 硬约束
- emoji 少量用（不超过 3 个/屏）
- 严禁暗沉、冷硬、"效率工具感"
- 严禁社交 App 元素（点赞 / 评论 / 关注）
- 存储层量词必须是"瞬间"不是"次"
- 所有文字从文案系统拷贝
- **整个 App 锁定浅色**，不要深色备选

## 输出
- 6 屏高保真 UI（浅色）
- 插画系统说明（元素 + 复用策略）

# 风格 3 · Korean Soft（韩系软系 · 新增）

## 为什么挑这个方向

- 和 Headspace 差别：没有儿童插画、更克制、更"成人可爱"
- 和 Zen 差别：柔软但不留白到 60%、允许一些色彩
- 情感承载力：中等，适合 Memorial mode（柔和能承接，但可能不够沉）
- 现实产品例子：Molly（韩国冥想）、Cake Resume、Toss（金融但视觉极软）、Kakao 生活类 App

## Dribbble 浏览入口

- 聚合 tag 页：
  - https://dribbble.com/tags/korean-aesthetic
  - https://dribbble.com/tags/korean_app
  - https://dribbble.com/tags/pastel-ui
  - https://dribbble.com/tags/soft-ui
- 推荐搜索的设计师 / 工作室：
  - `PLUS X` — 韩国顶级设计工作室
  - `Kakao` 关键词
  - `Hoonnie Cho`
  - `Jaeeun Ann`
  - `Sojung Lee`
  - `Naver` 关键词
- 关键词组合：`korean + mobile UI` / `pastel + ios` / `soft UI + app`
- 现实参考 App：Molly、Toss、Daily Routine、Habitify（韩国版）、Kakao Talk 的部分页面、iHerb（韩国市场版本）

## 视觉语言

**色彩（全浅色，糖霜系）**
- 主背景：#FAF7F2（奶油白，比奶咖浅）
- 卡片背景：#FFFFFF
- 次级背景：#F0EBE3（米杏色）
- 主文字：#2B2820（深暖灰）
- 次文字：#9A9488
- 时间账户色（稍亮一点的糖霜色）：
  - 陪父母 · 奶茶粉 #F2C5BA
  - 陪孩子 · 柠檬黄 #F5DB94
  - 陪伴侣 · 薰衣草紫 #C8B3E0
  - 运动 · 抹茶 #B5D4B5
  - 创造 · 天空蓝 #B5CFE0
- 所有色彩**高明度 + 中低饱和**（糖霜感，不是荧光）

**字体**
- 中文：Freesentation (韩国开源字体) 或 阿里妈妈方圆体 Medium
- 英文数字：Pretendard（韩国国民设计字体）、DM Sans
- 大数字：Pretendard 或 SF Pro Rounded Bold
- 对比强：标题 Bold、正文 Regular、不给 Light（韩国设计一般不太用极细字重）

**质感与图形**
- 非常柔和的阴影：offset 0 8 24 rgba(200,200,200,0.12)
- 圆角：卡片 20pt、按钮 16pt、chip 100pt（胶囊）
- 允许微妙的 inner shadow（inset 0 1 0 rgba(255,255,255,0.5)，模拟"糖霜"光泽）
- Icon：单色填充 + 圆角（不是纯线条，也不是插画）
- 允许微小的装饰元素：角标上的小圆点、卡片角的小图标、光晕

**布局**
- 中等密度，卡片之间 12-16pt 间距
- 数字居中对齐 + 大号字重
- 底部 Tab Bar 带填充 icon
- 状态栏下方大量使用"大标题 + 副标题 + 一行元信息"的三行结构

**动效**
- Spring response 0.45 / damping 0.8（类似 iOS 系统但更柔）
- 按钮按下 scale 0.97 + 阴影缩小
- 卡片浮起效果：hover 时上移 2px + 阴影变大
- 存入成功：柔和放大 + 颜色短暂深一档 + 回到原位

1. **Welcome** — 中央一个大号圆形插图（柔和渐变）+ 标题 + 按钮

3. **主页** — 顶部 greeting + 总账户卡（大号数字、糖霜阴影）+ 时间账户卡 grid（2 列）
4. **时间账户详情** — 顶部大号数字 + 参数卡（带填充 icon）+ 时间线
5. **新增时刻表单** — 糖霜感字段、9 宫格媒体圆角大
6. **时刻详情** — 媒体轮播圆角 20pt + chip 胶囊型 + 底部 floating 动作按钮

## 硬约束
- 饱和度必须 ≤ 40%（不能出现霓虹、荧光）
- 严禁卡通插画（Headspace 风格）
- 允许 emoji 一点点（0-1 个/屏）
- 存储层量词必须是"瞬间"

## 输出
- 6 屏高保真 UI

## 我对 5 个方向的诚实预测（不偏向）

| 方向 | 最大优点 | 最大风险 |
|------|---------|---------|
| **Warm Illustration** | 亲和力 A+，Persona 2（苏晴）型用户最爱 | 可能被误以为是"儿童向"或"习惯养成" |
| **Zen** | 调性契合度 A+，有独特美学 | 可能"太慢""太沉"，部分用户觉得压抑 |
| **Korean Soft** | 视觉讨喜 A，传播友好 | 可能看起来像"又一个浅色 SaaS"，缺少独特标志 |
| **Paper Journal** | 情感承载力 A+，差异化最强 | 工程实现纸纹理 / 折角 / 撕裂成本较高 |
| **Editorial** | Memorial mode 最适合，工程最简单 | 可能太冷、缺乏亲和力、Persona 2 不爱 |

3. **Zen** — 最有灵魂，但可能门槛高
4. **Editorial** — 最适合严肃场景，但可能失去一部分用户
5. **Korean Soft** — 稳妥但可能缺记忆点
