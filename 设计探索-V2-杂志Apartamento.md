## 1. 产品 DNA（30 秒）

时间银行 = 把"时间"当银行账户的 iOS App。

两层叠加：
- **消耗层**：自动算"和父母还能共度多少小时"等。数字一直在减少，但不是焦虑倒计时——是温柔提醒"现在很值钱"。
- **存储层**：用户主动把"有意义的瞬间"（带图文）存进对应账户。这些被存入的内容**永不被消耗**。

哲学：《四千周》(Oliver Burkeman) + 银行账户隐喻。

调性：**克制 · 温暖 · 不焦虑 · 让人每天想点开看一下**。

## 2. 核心情绪关键词

**永远写**：温柔 / 留白 / 让人慢下来 / 像一封信 / 不打扰

**永远不写**：

## 4. 风格 · 杂志高级风 · Apartamento / Kinfolk / Cereal

### 整体氛围（核心 · 自己感受）

> 周日上午十点。哥本哈根公寓的厨房，木桌，一杯黑咖啡，桌上摊开一本 Apartamento 季刊。光从大窗斜进来，纸是哑光的、有微微的 cream 色。你慢慢翻，一页一页。

参考锚（**这是关键 · 请仔细去看**）：
- **Apartamento Magazine**（米兰家居刊 · apartamentomagazine.com）—— 大字标题 + 摄影 + 大量留白
- **Kinfolk Magazine**（北欧 lifestyle）—— 字体的极致克制
- **Cereal Magazine**（伦敦旅行）—— 极简 editorial 的天花板
- **Wallpaper\* Magazine**（设计 / 建筑）—— 现代奢华
- **The Gentlewoman**（女性人文）—— 大粗 serif hero + 优雅留白
- **Aesop 包装 + 网站** —— 同一精神在 commercial 端的体现
- **Snow Peak 网站** —— 季刊排版的 web 版

### 关键情绪锚点

- 「**slow magazine**」—— 季刊不是周刊，节奏是慢的、值得反复翻的
- 「翻页节奏」—— UI 不像 App，像翻书
- 「**字体 hero**」—— 大粗 serif 标题 / 数字是视觉主角
- 「裸色 + 杂志摄影」—— 不是糖果色不是科技色
- 「印刷感」—— 字间距、行距讲究，纸感（不是玻璃感）

### 颜色 · 给方向不给 HEX

往 **燕麦白 / 裸色 / 米色背景 + 砖红 / 锈红 / 深绿 / 海军蓝**走。Accent 可以是单一深色（不是亮色）。**避免**糖果色、霓虹色、纯白。**纸感而不是屏幕感**——背景不要纯白，要 off-white 偏暖。

### 字体 · 给方向不给字体名

主标题 + 数字用**大粗 display serif**（GT Sectra Display / Domaine Display / Saol Display / Tiempos Headline / 思源宋体 Bold）。这是核心 —— **数字必须用 serif，是杂志风的灵魂**。

正文 / 副文案用现代 sans（Söhne / Inter / Neue Haas Grotesk / 思源黑体）。

可加 italic serif 作为引文 / 副标题装饰（Apartamento 风），让排版有"杂志栏目感"。

### 卡片 / 容器 / 装饰

- 卡片像翻开的杂志双页 —— 有"印刷感"，不是玻璃
- 用**真正的栏目排版**：标题 / 副标题 / dropcap / 引文 / 页码
- 圆角小（4-8pt）甚至方角，强化"页"感
- **可以**有极淡的纸纹理（noise 5-8%）
- **不要**毛玻璃、渐变、微动效

### Lifespan 卡 · 必须出 2-3 个变体（重点）

- **变体 A · 杂志双页**：左页标题"时间余额" + 大粗 serif 数字"2,288"; 右页标题"已存入" + 数字"128 小时"。中间一道极细 hairline 像装订线。整卡像翻开的双页对开。
- **变体 B · 杂志专题块**：上半像 Cereal 的章节扉页 —— 一行 sans caps small "TIME BALANCE" + 大粗 serif 数字 hero 占据 60% 高度 + 下方一行细斜体引文"约 44 年 · 386 Kh"。下半同结构对应"已存入"。
- **变体 C · 章节式诗意**：把数字嵌入文学化句子 —— 大粗 serif 主行"还有 **2,288** 周可以慢慢用。" 数字粗 italic 用 accent 色突出。下方"已经留下了 **128** 小时。" 整卡像 Apartamento 主编的开篇语。

竖向并列，label "变体 A / B / C"。

### 整体感

像 Apartamento 第 35 期某篇专题。每数字都被精心放置过，每留白都有意图。读完不焦虑，反而想再翻一遍。

## 7. 不要做的事

- ❌ NYT 那种新闻杂志（信息密度高 / 严肃）—— 这次走的是 lifestyle slow magazine
- ❌ 像 Pinterest 风格的"杂志感模板"（千篇一律）—— 必须有独特栏目编排
- ❌ 糖果色 / 霓虹 / 高饱和（杂志风是裸色 + 深色）
- ❌ 玻璃 / 渐变（这是印刷感不是屏幕感）
- ❌ Onboarding / 设置等辅助页
- ❌ Logo / icon 设计
- ❌ 徽章 / 进度条
- ❌ Memorial mode
