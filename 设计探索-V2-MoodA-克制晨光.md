## 1. 产品 DNA（30 秒读完）

**时间银行**是一个 iOS App，把"时间"当成银行账户来使用。

它有两层叠加：

- **消耗层**：自动算出你和父母/孩子/伴侣"还能共度多少小时"、运动/创造/自由"还剩多少时间"。这些数字会一直减少，但不是焦虑倒计时——是**温柔提醒"现在很值钱"**。
- **存储层**：用户可以手动把"有意义的瞬间"（带图文）存进对应的时间账户。这些被存入的内容**永不被消耗、永不被时间带走**。

每个时间账户卡片同时显示两个数字：「还能共度 552h」+「已存入 56h」。

哲学源头：《四千周》(Oliver Burkeman) + 银行账户隐喻。**核心不是"剩余倒计时焦虑"，而是"被感受到的时间不会被带走"**。

调性：**克制 · 温暖 · 不焦虑 · 让人每天想点开看一下**。

## 4. Mood「克制晨光」（本次唯一 mood · 严格执行）

> 灵感参考：无印良品 / Notion / 早安手账 / 日本 LIFULL HOME'S
>
> 关键词：极简日系 / 浅米色底 / serif 标题 / 大量留白 / 等宽数字

### 色板（必须用这组，不要替换）

| Token | 色值 | 用途 |
|---|---|---|
| 背景 | `#FAF7F1` | 整体背景（奶白）|
| 卡片 | `#FFFFFF` | 卡片表面 |
| 分隔线 | `#EDE5D6` | hairline / divider |
| 文字主 | `#2A2A26` | 标题、大数字 |
| 文字次 | `#6B6862` | 副文案、元信息 |
| 文字三 | `#A8A39A` | 占位文字、tertiary label |
| Accent 珊瑚橘 | `#E89A7A` | **仅用于强调**，每屏不超过 2 处（如 FAB 按钮、当前 Tab、关键 chip）|

> **饱和度低，避免任何明亮色**。即便是 Accent 也是低饱和。

### 字体

| 用途 | 推荐字体 |
|---|---|
| 标题 | 思源宋体 Light / Times New Roman / Cormorant Garamond / Noto Serif SC Light |
| 正文 | 思源黑体 Regular / Inter / -apple-system / Noto Sans SC |
| 数字（关键）| **JetBrains Mono / SF Mono / IBM Plex Mono 等宽字体** —— 让 552h、128h 这种数字像账本里的记账数字 |

### 卡片视觉

- 极轻投影：`box-shadow: 0 2px 8px rgba(0,0,0,0.04)` 或干脆只用 `1px solid #EDE5D6` hairline
- 圆角 12-16
- 卡片之间间距大（16-24pt），留白多
- 双数字卡内部：左右两列用 1px hairline 分隔，不用大面积填色

### 排版细节

- 标题字号区分明显（大标题 28pt / section 16pt / body 14pt）
- 行间距宽（line-height 1.5-1.7）
- 中文标点用全角，英文/数字之间空格留半角

### 整体感

像一本米色道林纸装订的手账，每一页都让人想慢下来翻。**克制是核心**——少即是多，留白比内容更重要。

### Lifespan 卡变体设计提示（本次重点）

请基于本 mood 输出以下 2-3 个变体（不要混入其他 mood 元素）：

- **变体 A 保守**：浅米色卡片 + serif 主数字（Cormorant Garamond Bold 或思源宋体 Light · 60-72pt），等宽副文案（JetBrains Mono · 13-14pt），上下半中间用 1px hairline `#EDE5D6` 分隔。珊瑚橘 accent 仅用于"周"或"已存入"label 一处提色

- **变体 B 创新 · 账本一页**：把双层卡设计成"账本翻开的一页"——左对齐主数字像账目记录，加一条细金线作为页眉（替代 hairline），"约还剩"用衬线小字像章节标题悬浮在数字上方。整卡有极轻微的纸纹 noise（5% opacity）

- **变体 C 大胆 · 极致俳句**：卡片几乎只有「2,288」和「128」两个 hero 数字（80-100pt serif），副文案缩到极小一行（11pt 等宽）。整卡留白超过 60%，看起来像一首俳句留白构图，让数字本身成为构图主体
