## 这一版本的风格定位
「Refined Minimalism · 印刷品般的克制」

灵感参考（都研究一下）：
- Claude.ai 官网的页面排版
- Linear.app
- Arc Browser 的书签管理界面
- Notion Calendar
- The Browser Company 的文字处理
- Eric Gill、Jan Tschichold 的书籍版式
- 意大利高端期刊 Apartamento

## 视觉语言

**色彩（浅色 + 深色两套，V1 只深色）**
- 深色底：#0F0E0C（暖炭色，不是纯黑）
- 浅色卡片背景：#1A1916
- 主文字：#E8E3D8（做旧米白）
- 次文字：#8A8277
- 暖强调色：#C9A961（黯淡金，只用在极少地方）
- 冷强调色：#7A8C9E（铅灰蓝）
- 语义色（维度）用低饱和的大地色系（褐、赭、灰、苔绿、旧蓝），不要任何鲜艳色

**字体**
- 中文：思源宋体（Source Han Serif）—— Display 场景用
- 英文数字：Newsreader / Ivar / Söhne Mono / Editorial New —— 所有大数字必须是衬线字体
- 次要文字：PingFang SC / Inter —— 走 Light 300 字重
- **大数字一律 ExtraLight / Light 字重**，禁止 Medium 以上

**质感与图形**
- 纸感、印刷感；允许微小的 grain 噪点（0.5% 不明显的纸纹）
- 线条全部用 hairline（0.5px 或 1px）
- 分隔线不用灰色横条，用"留白 + 字号对比"来区分层级
- 无阴影、无渐变、无毛玻璃
- 图标只用细线条单色，禁止任何填充色块

**布局**
- 留白极大，至少 40% 面积留白
- 每一屏只有一个视觉焦点
- 文字左对齐为主，大数字居中
- 底部 Tab Bar 极细、几乎不可见（只有 icon，无文字）

**动效**
- 克制，只有 fade + 微小上移（最多 4px）
- 禁止 spring 弹性效果
- 存入成功动效：一滴光从中心慢慢消隐到右上角（800ms 线性）

## 请设计以下 6 屏（iPhone 16 Pro 尺寸 393×852）

1. **Welcome 启动页** — 极简标语 + 一个按钮
2. **Onboarding Step 2（关系 chip 多选）** — 4 个 chip 让用户选"父母健在 / 有孩子 / 有伴侣 / 独处也是一种时间"，页面顶部问句"你和谁共度着时间？"
3. **主页** — 顶部总账户卡 + 3-4 张双数字维度卡（陪父母 552h / 56h、陪孩子、陪伴侣、创造）+ 底部 Tab（主页/账户/设置）
4. **维度详情（陪父母）** — 上半大数字 "552h"（衬线字体）+ 参数卡，下半是时间线（3 条时刻，缩略图 + 标题 + 时间）
5. **新增时刻表单** — 维度/时间/时长/标题/笔记/9 宫格媒体 + "存入时间银行"按钮
6. **时刻详情（"杭州西湖 48 小时"）** — 顶部媒体轮播（模拟一张风景照）+ 标题 + 元信息 chip + 笔记文字 + 底部 3 个动作按钮

## 硬约束（不能违反）
- 禁用 emoji（除了极少必要的 UI 符号，如 ♡ ◐）
- 禁止使用红色（除了删除按钮）、鲜艳色、渐变彩色
- 禁止任何"打卡/成就/徽章/进度条/完成率"视觉语言
- 禁止圆角超过 16pt（保持印刷品般的方正感）
- 所有文字从「文案系统」里拷贝，不要自己写文案

## 输出
- 6 屏高保真 UI 设计
- 每屏标注色号、字号、间距
- 简短设计理念说明（300 字内）
- 一个可复用的 design tokens 表（颜色、字体、间距、圆角）

```
你是一位擅长治愈系插画 UI 的设计师，为关系和情感类产品做视觉。

## 产品背景

「Warm Illustration · 如抱着一只猫的下午」

灵感参考：

- Finch（自我关怀 App）
- Fabulous（习惯养成 App）
- Japanese picture book 插画（Kumon、福音馆书店）
- 韩国插画家 Puuung 的"爱就在细节里"系列
- Pinterest 搜 "soft UI illustration 2024"

## 视觉语言

**色彩（温暖治愈系）**
- 主背景：#FBF4EC（奶咖色）或 #2E2520（深巧克力，深色模式）
- 珊瑚橘：#E89A7C（父母维度）
- 金黄：#E8C47C（孩子维度）
- 薄荷绿：#8EC7A8（运动）
- 晨雾蓝：#A8C4D9（创造）
- 夕阳紫：#C29DD4（伴侣）
- 所有色彩饱和度不超过 60%，给人温暖不刺眼

**字体**
- 中文：阿里普惠 Medium / OPPO Sans Medium / 站酷庆科黄油体
- 英文数字：DM Sans / Poppins / Fraunces（如需 serif）
- 不走 Light 字重，**Medium 500 是默认**
- 大数字可以用 Display 级字重 Semibold，但配圆润字形

**插画与图形**
- 每个维度 icon 是一张小插画而不是符号：陪父母画成两个坐在长凳上的背影剪影，陪孩子是一只小手在牵大手，等等
- 允许使用情绪化符号：温柔的眼睛、伸开的手掌、抱着的人、光晕
- 圆角全部 20pt+，卡片 24pt，按钮 100pt（胶囊）
- 允许渐变（温暖方向的，如橙→粉→黄），禁止冷色渐变
- 按钮阴影允许（柔和暖色，offset 0 4 12）
- 插画用细线条 + 大色块组合（不是纯线条，也不是纯扁平）

**布局**
- 中等密度，不需要极端留白
- 主色块大、可以铺满卡片背景
- 文字可以居中
- Tab Bar 明显可见，有插画式 icon

**动效**
- Spring 弹性明显，damping 0.7-0.8
- 按钮按下有"呼吸"感（scale 0.95 带回弹）
- 存入成功动效：一颗心形光点 + 周围放射微光 + 轻微震动反馈
- Onboarding 每屏之间用 slide + scale 组合过渡

## 请设计以下 6 屏（iPhone 16 Pro 尺寸 393×852）

1. **Welcome 启动页** — 一幅温暖插画（两人坐在窗前）+ 标题 + 按钮
2. **Onboarding Step 2** — 4 个 chip（带小插画 icon）"父母健在 / 有孩子 / 有伴侣 / 独处也是一种时间"
3. **主页** — 顶部总账户（带光晕动画暗示）+ 维度卡片（每卡片左侧一个小插画 + 右侧双数字）
4. **维度详情（陪父母）** — 顶部大插画（两位老人剪影）+ 大数字 "552h" + 参数卡 + 下方时间线
5. **新增时刻表单** — 字段带可爱 placeholder 插画
6. **时刻详情** — 媒体轮播 + 温暖的 chip + 笔记文字排版自由（允许手写感）

## 硬约束
- emoji 可以少量用（不要超过 3 个/屏）
- 严禁暗沉、冷硬、"效率工具感"
- 严禁社交 App 常见元素（点赞 ❤️ / 评论 / 关注）
- 存储层量词必须是"瞬间"不是"次"
- 所有文字从文案系统拷贝

## 输出
- 6 屏高保真 UI
- 插画系统设计说明（有哪些元素、如何复用）
- design tokens 表
- 一个可爱但克制的 brand 情绪板（3 张配图）

## 视觉语言

**色彩（使用 iOS 系统语义色）**
- systemBackground / secondarySystemBackground / tertiarySystemBackground
- label / secondaryLabel / tertiaryLabel
- systemRed / systemOrange / systemYellow / systemGreen / systemMint / systemTeal / systemCyan / systemBlue / systemIndigo / systemPurple / systemPink

**字体**

**图形与质感**

- 允许使用 Toolbar 底部按钮组

**动效**
- 完全使用 iOS 系统标准过渡（navigation push、sheet slide up）
- Spring response 0.5 / dampingFraction 0.825（默认系统 spring）
- 存入成功动效：类似 Apple Fitness 圆环填充的感觉

3. **主页** — NavigationStack + ScrollView，顶部总账户用大号 SF Rounded 数字，维度卡片用 grouped list + 毛玻璃
4. **维度详情** — Large title + 数字 display + Form-like 参数卡 + 下方时间线
5. **新增时刻表单** — 标准 Form，每行一个字段（List style grouped insetted）
6. **时刻详情** — 全屏媒体 + 下方 grouped list 式元信息 + Toolbar 底部操作按钮

- 严禁"强视觉冲击"的大色块
- 所有交互必须符合 HIG
- 存储层量词必须是"瞬间"

## 风格 4 · Eastern Zen（禅意东方 / 墨与宣纸）

```
你是一位深谙东方美学的 UI 设计师，擅长把禅意注入现代产品。

我在做一款 iOS App 叫「时间银行」——让用户同时看到和父母/伴侣/孩子还能共度的小时数（消耗层），并把有意义的时刻带图文永久存进账户（存储层）。

这个产品本质上讨论的是"时间的有限" vs "当下的珍惜"——这是东方禅宗最擅长的命题。我想做一版以禅意美学承载这份重量的视觉。

「Eastern Zen · 如一盏茶后的静」

灵感参考：
- 无印良品（官网、App、产品包装）

- 台湾诚品生活
- Muji to Go / Muji Hotel 的内部导视系统
- 日本摄影师川岛小鸟 / 滨田英明的色调

## 视觉语言

**色彩（极度克制，基本只用 5 色）**
- 宣纸白：#F5F0E8（浅色模式主背景）
- 墨色：#1A1A1A（深色模式主背景 / 浅色模式主文字）
- 远山灰：#8B8680（次文字）
- 松烟色：#373330（次级背景）
- 朱砂红：#B14545（只用于删除等极少场景）
- 竹青：#6B8570（只用于一个维度的强调）
- **整个 App 可能只有 3 种颜色**

**字体**
- 中文：筑紫 A 丸明朝 / 思源宋体 Heavy 对比 ExtraLight / 汉仪尚巍手书（签名感）
- 英文数字：Ivar Display / Söhne Breit / Untitled Serif（古朴衬线）
- 大数字一律 ExtraLight，看起来像蘸墨写的一笔
- 允许纵向排版（少数标题处，日文书籍式）

**质感与图形**
- 纸感：0.8% 的不均匀纸纹噪点，微微发黄
- 所有线条用"细+粗"对比（不是均匀的 1px），模拟毛笔
- 允许微小的水墨晕染效果（卡片边缘有若有若无的 0.3% 晕开）
- 圆：允许使用圆形作为主图形元素（月、窗、禅圆）
- 禁止纯几何硬直角；所有边角都略带"手绘不完美"
- 插画只用水墨线条，无填充色

**布局**
- **留白占据 60%+ 的屏幕**
- 允许使用日式排版的三分结构

### 推荐策略

**"All-in 一个方向"**：选得分最高的一个，完全按照它来做 V1。这是最清晰的做法，但如果选错后期返工成本高。

**"主方向 + 细节借用"**：选一个主方向（建议 Refined 或 Zen），然后从其他方向借一些具体元素（比如借 Meditative 的"呼吸动效"、借 Headspace 的"维度插画"）。

**我的个人预测**（仅供参考，不带偏见）：
- **最稳妥**：Refined Minimalism（Claude 风）——工程最简单、最不容易过时
- **最感人**：Zen（禅意）——和产品主命题最契合，但可能"太慢"让部分用户觉得沉重
- **最易传播**：Headspace 风——亲和力强，小红书用户易接受
- **最 Apple Design Award 候选**：Meditative Calm——视觉最独特，但工程最难
- **最保险**：Apple 风——一定不出错，但可能缺少灵魂
