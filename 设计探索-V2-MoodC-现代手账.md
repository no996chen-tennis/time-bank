## 1. 产品 DNA（30 秒读完）

**时间银行**是一个 iOS App，把"时间"当成银行账户来使用。

它有两层叠加：

- **消耗层**：自动算出你和父母/孩子/伴侣"还能共度多少小时"、运动/创造/自由"还剩多少时间"。这些数字会一直减少，但不是焦虑倒计时——是**温柔提醒"现在很值钱"**。
- **存储层**：用户可以手动把"有意义的瞬间"（带图文）存进对应的时间账户。这些被存入的内容**永不被消耗、永不被时间带走**。

每个时间账户卡片同时显示两个数字：「还能共度 552h」+「已存入 56h」。

哲学源头：《四千周》(Oliver Burkeman) + 银行账户隐喻。**核心不是"剩余倒计时焦虑"，而是"被感受到的时间不会被带走"**。

调性：**克制 · 温暖 · 不焦虑 · 让人每天想点开看一下**。

## 4. Mood「现代手账」（本次唯一 mood · 严格执行）

> 灵感参考：iOS 26 Liquid Glass / Linear / Arc Browser / Things 3 / 旧版 macOS Big Sur Control Center
>
> 关键词：Apple 玻璃 / 柔和渐变 / 立体感 / 现代精致 / 朝阳

### 色板（必须用这组，不要替换）

| Token | 色值 | 用途 |
|---|---|---|
| 背景渐变 | `linear-gradient(135deg, #E8E5F2 0%, #F5EFE8 100%)` | 整体背景（左上浅蓝紫 → 右下暖米白，类似日出方向）|
| 卡片底色 | `rgba(255,255,255,0.72)` + `backdrop-filter: blur(20px)` | 半透明毛玻璃 |
| 卡片边框 | `1px solid rgba(255,255,255,0.5)` inset | 让卡片浮起 |
| 文字主 | `#1D1B25` | 标题、大数字 |
| 文字次 | `#6E6B78` | 副文案、元信息 |
| 文字三 | `#B0AEB8` | 占位文字、tertiary label |
| Accent 朝阳橘 | `#FF9F6B` | **仅 lifespan 上半数字 + FAB 按钮**（核心识别物，不滥用）|

> **柔和、不亮**。渐变饱和度低，毛玻璃透明度温和，整体像清晨刚拉开窗帘的光感。

### 字体

| 用途 | 推荐字体 |
|---|---|
| 标题 | **SF Pro Display Bold / 思源黑体 Bold / Inter Bold** |
| 正文 | SF Pro Text Regular / 思源黑体 Regular |
| 数字（关键）| **SF Pro Rounded** —— 圆润现代感，比标准 SF Pro 更有手账感 |

### 卡片视觉

- 半透明白 + `backdrop-filter: blur(20px)` 营造毛玻璃
- **圆角大**：卡片 20-24，按钮 12-16
- 微妙阴影：`box-shadow: 0 8px 24px rgba(29,27,37,0.06), 0 1px 0 rgba(255,255,255,0.5) inset`
- 卡片之间用渐变背景分隔，不用 hairline
- 双数字卡内部：左右两列用 `1px solid rgba(29,27,37,0.06)` 极细分隔

### 排版细节

- 数字大且粗：lifespan 主数字 60-72pt SF Pro Rounded Bold，时间账户卡数字 32-40pt
- 行间距 1.4-1.5
- 中文标点用全角，英文标点 + 数字用半角
- Section header 用小字 + 大写 letter-spacing（如 "TIME ACCOUNTS · 6"）

### 整体感

像 iOS 26 system app 的"放大版"——毛玻璃的精致感 + 渐变的温暖光感 + 圆润字体的手账感。**核心是"立体光感"**——让用户感觉每张卡片都浮在背景上，像清晨雾气里看见的远山。

> 关键边界：**不要做成 Spotify 那种深色玻璃**——背景是浅亮的渐变，玻璃是浅白半透明，整体明度高。

### Lifespan 卡变体设计提示（本次重点）

请基于本 mood 输出以下 2-3 个变体（不要混入其他 mood 元素）：

- **变体 A 保守 · 单层玻璃**：单张毛玻璃白卡 `rgba(255,255,255,0.72)` + `backdrop-blur(20px)` + 1px inset 白边。上下两半中间一条 `1px solid rgba(29,27,37,0.08)` 极细分隔。主数字 SF Pro Rounded Bold 60pt `#1D1B25`，朝阳橘 `#FF9F6B` accent 仅用于"周"字符 + 上方一个微小 sun 图标

- **变体 B 创新 · 双层玻璃叠卡**：上下两层独立玻璃卡，错位 8px 叠加形成深度——上层浅紫玻璃 `rgba(232,229,242,0.6)` + 下层浅米玻璃 `rgba(245,239,232,0.7)`，每层都有独立 backdrop-blur。叠加处一道极细朝阳橘渐变光（`linear-gradient(90deg, transparent, #FF9F6B40, transparent)`），让两层之间像有一缕晨光穿过

- **变体 C 大胆 · 周数光点矩阵**：把 4400 周（85 岁人生总周）渲染成 4400 个微小光点 (2px 圆) 排列成矩阵——已逝 2,112 周用 `#B0AEB8` 灰、剩余 2,288 周用 `#FF9F6B` 朝阳橘渐变。主数字「2,288」大字 (SF Pro Rounded Black 72pt) 半透明浮在光点之上 (`opacity 0.92` + `mix-blend-mode: multiply`)。下半「已存入」叠在矩阵下方一条独立玻璃带。**这是给那些被《四千周》打动的用户的彩蛋**
