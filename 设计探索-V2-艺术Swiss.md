## 1. 产品 DNA（30 秒）

时间银行 = 把"时间"当银行账户的 iOS App。

两层叠加：
- **消耗层**：自动算"和父母还能共度多少小时"等。数字一直在减少，但是温柔提醒"现在很值钱"，不是焦虑倒计时。
- **存储层**：用户主动把"有意义的瞬间"（带图文）存进对应账户。**永不被消耗**。

哲学：《四千周》(Oliver Burkeman) + 银行账户隐喻。

调性：**克制 · 温暖 · 不焦虑 · 让人每天想点开看一下**。

## 4. 风格 · 设计艺术 · 瑞士平面 / Swiss

### 整体氛围（核心）

> 1960 年的苏黎世。Müller-Brockmann 工作室。墙上一张精确到毫米的网格海报，桌上一杯黑咖啡。所有东西都有理由存在的位置。**理性、克制、有点冷峻，但不无情**。

参考锚（**关键 · 请仔细看**）：
- **Josef Müller-Brockmann** —— 网格系统之父，所有海报作品
- **Massimo Vignelli** —— 美铁标牌系统 / Knoll / Heller 餐具
- **Helmut Schmid** —— 朝日啤酒包装 / 排版书《Typography Today》
- **MoMA App** —— 现代主义在数字端的当代延伸
- **Pentagram 出品**（Paula Scher / Michael Bierut）—— 现代瑞士派
- **Dieter Rams + Braun 产品手册**
- **Akzidenz-Grotesk / Helvetica / Univers 字样海报**
- **Müller-Brockmann《Grid Systems in Graphic Design》**

### 关键情绪锚点

- 「**精确**」—— 每个元素都对齐到网格，毫米级
- 「**Helvetica 灵魂**」—— 中性、克制、不张扬
- 「**Information Design**」—— 数据是主角，装饰是佣人
- 「**留白即结构**」—— 不是装饰性的留白，是网格定义的间距
- 「**冷而不僵**」—— 这是关键，瑞士派可以理性但不能没人味

### 颜色 · 给方向不给 HEX

**纯黑 + 纯白 + 一抹高饱和 accent**（鲜红 / 鲜蓝 / 鲜黄 / 荧光绿，**任选一个**）。Helvetica + 鲜红是经典组合（Vignelli 美铁），任何 mood 不可有第二个 accent。**避免**中性灰、米色、任何"温暖色"——这是瑞士派的核心冷感。

### 字体 · 给方向不给字体名

**全部 sans-serif，无衬线**。主推 Helvetica Now / Akzidenz-Grotesk / Univers / Inter / GT America / Aktiv Grotesk。**所有字号都用同一字体的不同 weight**——这是瑞士派核心：用 weight + size 建立层级，不用换字体。

数字也用同一 sans，**绝对不要 serif**（serif 数字是杂志风的事，这里不是）。

### 网格 · 这是 Swiss 的灵魂

主页强 12-column 网格。**所有元素必须对齐**。不要随意 padding，所有间距都是网格 unit 的倍数。卡片之间的间距 = 网格列宽 × N。**让网格可见性有时露出**（如让某些 hairline 跟列对齐）。

### 卡片 / 容器 / 装饰

- 卡片可以**根本没有"卡片"概念**——只用网格 + hairline 划分区域
- 圆角 = 0（直角）或极小（2pt）
- **不要**任何装饰元素：图标弱化到几何形 / 不要插画 / 不要 emoji
- 可用 1px 高对比 hairline（黑或 accent 色）作为分隔

### Lifespan 卡 · 必须出 2-3 个变体（重点）

- **变体 A · 经典模数**：12-column 严格对齐 + Helvetica Bold 数字 + 高对比黑白 + 唯一 accent 色用于"周"或某个关键字符。整卡看起来像 Müller-Brockmann 1965 年的演讲海报。
- **变体 B · Information Design**：把 lifespan 数据可视化为一个**精确条状图**——把 4400 周（85 岁人生总周）画成一条横向条，已逝部分用灰填充，剩余 2,288 周用 accent 色填充，主数字"2,288"以 Helvetica Bold 浮在条上方对齐。下半同结构对应"已存入"。这是 Vignelli 风的 InfoDesign。
- **变体 C · 极致几何**：把 lifespan 卡解构成纯几何 —— 一个大方块（accent 色）+ 数字嵌入方块边缘 + Helvetica Bold 大字。下半"已存入"是另一个对位的方块。整卡像 De Stijl 蒙德里安的当代复刻。

竖向并列，label "变体 A / B / C"。

### 整体感

像走进 MoMA 现代主义展厅。每一寸都被设计师权衡过、对齐过、计算过。**不是冷漠，是被尊重**。

## 7. 不要做的事

- ❌ Serif 字体（这次只 sans）
- ❌ 米色 / 暖色 / 中性色 → 必须高对比黑白 + 一个 accent
- ❌ 圆角大、毛玻璃、渐变、噪点纹理
- ❌ 装饰性 illustration
- ❌ Onboarding / 设置等辅助页
- ❌ Logo / icon 设计（用极简几何形或 SF Symbols 占位）
- ❌ Memorial mode
- ❌ "温暖治愈"风格的卡通 emoji ❌
