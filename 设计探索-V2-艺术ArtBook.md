## 1. 产品 DNA（30 秒）

时间银行 = 把"时间"当银行账户的 iOS App。

两层叠加：
- **消耗层**：算"和父母还能共度多少小时"等。温柔提醒"现在很值钱"。
- **存储层**：手动把"有意义的瞬间"存进对应账户，永不被消耗。

哲学：《四千周》+ 银行账户隐喻。

调性：**克制 · 温暖 · 不焦虑 · 让人每天想点开看一下**。

## 4. 风格 · 设计艺术 · Art Book / 装帧

### 整体氛围（核心）

> 大英图书馆地下室。一本 1856 年精装初版的羊皮卷，被小心翻开放在木台上。象牙色纸页，铜版印刷的章节符，扉页有手写的题跋。**沉静、庄严、有重量**。

参考锚（**关键 · 请仔细看**）：
- **TASCHEN 出版物**（特别是大开本艺术全集）
- **Phaidon 出版物**（艺术家专著系列）
- **Steidl 出版物**（精装摄影书）
- **苏富比 / 佳士得拍卖目录**
- **Penguin Classics 黑色精装系列**
- **大英博物馆图录**
- **Bauhaus 1919 manifesto 印刷版**
- **William Morris 工艺美术运动印刷品**
- **Aldine Press 文艺复兴印刷**

### 关键情绪锚点

- 「**精装书的重量**」—— 不是手机感，是书页感
- 「**章节符 § ¶ †**」—— 古典印刷的节拍器
- 「**字号反差极大**」—— 巨大章节编号 + 微小注释脚
- 「**dropcap / 首字下沉**」—— 段落起始的仪式感
- 「**对开版 facing pages**」—— 双层结构利用页面对位
- 「**羊皮 / 铜版 / 蜡封**」—— 物质感暗示，但要现代化
- **不是中国古籍 / 不是日本和本** —— 这是欧洲精装书传统

### 颜色 · 给方向不给 HEX

**象牙白 / 哑米白纸色 + 浓墨黑 + 一抹深红或绛蓝**（象征装订线 / 章节标记 / 红字注疏）。可加极少量金色作为章节符。**饱和度低**，所有颜色都像被时光磨过。

### 字体 · 给方向不给字体名

**核心：古典 serif** —— Caslon / Bembo / Garamond / Mrs Eaves / Sabon / 思源宋体 / Playfair Display Italic。**章节编号可加 caps small + letter-spacing**。可以混用 ScalaSans / Trade Gothic 作为脚注 / 元数据。

数字也用 **classic serif**（不是现代 sans）。

### 卡片 / 容器 / 装饰

- "卡片"概念**淡化** —— 用书页栏目（column / margin / running header / page number / chapter rule）替代
- 圆角 = 0（书页是方的）
- 可以加：极轻噪点纸纹 / 装订线（左侧极细 hairline）/ 章节符 / dropcap / footnote rule
- **可以**用 ornament（细小装饰花纹）作为分隔，但要克制

### Lifespan 卡 · 必须出 2-3 个变体（重点）

- **变体 A · 精装扉页**：上方一行 caps small "CHAPTER ONE" 或"§ I" 章节标记 + 下方大字古典 serif "2,288 周" + 极细 ornament 分隔线 + 下半 "Chapter Two · 已存入 128 小时" 同样格式。整卡像翻开的精装书章节首页。
- **变体 B · 拍卖目录**：模仿苏富比目录的拍品介绍 —— 大字 lot number 风格"NO. 2,288 周" + 小字 italic 元信息"约 44 年 · 386 Kh, 林晚 (b. 1985)" + 下半同样规格"NO. 128 已存入". 整卡像图录里一页珍贵藏品。
- **变体 C · 古帖题跋**：把数字嵌入古典印刷的复杂排版 —— dropcap 装饰首字 "约" 下沉 4 行 + 后续文字"...还剩 2,288 周可慢慢用。" 一段段排版像古籍引文，章节符号、page number、 footnote 全部具备。整卡像一段精装书的开篇 manuscript。

竖向并列，label。

### 整体感

像翻开一本 19 世纪由 William Morris 设计的精装书。每一页都被装帧师细心规划过。**博物馆级的庄严 + 古典 typography 的细节克制**。

## 7. 不要做的事

- ❌ 现代极简（这是 19 世纪精装书 + 当代 App，不是 Notion）
- ❌ Sans-serif 主字体（必须古典 serif）
- ❌ 圆角大、毛玻璃、渐变
- ❌ 中国古籍装帧（线装、蝴蝶装等）—— 这是欧洲精装传统
- ❌ 数字字体（数字必须 serif）
- ❌ Onboarding / 设置等辅助页
- ❌ Logo / icon 设计 → 用古典 ornament 占位
- ❌ Memorial mode
