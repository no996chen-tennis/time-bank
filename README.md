# 时间银行（Time Bank）

一款 iOS App 的产品文档仓库。

**产品一句话：** 把有意义的时间，存进来。

**产品两句话：** 时间在流逝，但被感受过的时间可以留下。时间银行让你看到自己和家人/伴侣/孩子还能共度多少小时，也让你把那些真正重要的瞬间（带着照片、视频和文字）**永久存入**你的账户。

**状态：** 产品定义与设计阶段（V1 尚未开发）

## 产品核心概念（30 秒版）

时间银行有**两层**：

**1. 消耗层（Counter Layer）**
时间在不可逆流逝。每张维度卡显示"还能存入 552h · 约 92 次见面"——自动计算，无需用户输入。

**2. 存储层（Deposit Layer）**
用户主动把有意义的时刻"存进"某维度。带图文视频，**永久封存，不会被扣减**。卡片同时显示"已存入 56h · 12 个瞬间"。

两层在每张维度卡底部形成对照：一个在减少、一个在增加。这是 App 叫"时间银行"的根本原因。

## 技术栈（规划）

- SwiftUI + WidgetKit（iOS 17+）
- SwiftData 本地存储 + FileManager 沙盒存媒体
- V1 纯本地无后端；V1.5 CloudKit 同步
- 无任何第三方 SDK、无账号体系、无数据收集
