## 1. 产品 DNA（30 秒）

时间银行 = 把"时间"当银行账户的 iOS App。

两层叠加：
- **消耗层**：自动算"和父母还能共度多少小时"等。数字一直在减少，但不是焦虑倒计时——是温柔提醒"现在很值钱"。
- **存储层**：用户主动把"有意义的瞬间"（带图文）存进对应账户。这些被存入的内容**永不被消耗**。

哲学：《四千周》(Oliver Burkeman) + 银行账户隐喻。**核心不是焦虑倒计时，是"被感受到的时间不会被带走"**。

调性：**克制 · 温暖 · 不焦虑 · 让人每天想点开看一下**。

## 2. 核心情绪关键词

**永远写**：温柔 / 留白 / 让人慢下来 / 像一封信 / 不打扰

**永远不写**：

- ❌ KPI 进度条 / 徽章 / 成就感
- ❌ 强 saturation / 闪光 / 微动效

## 4. 风格 · 禅意 · 日式无印

### 整体氛围（核心 · 自己感受）

> 早晨 9 点，京都 muji 实体店里的样品桌。原木桌面，米白色的笔记本，桌角一杯凉了一半的麦茶。光是斜的、温的、薄的。你不忙，但在认真。

参考锚（自己去看 + 自己提炼）：
- **MUJI 官网 / muji.com** —— 产品摄影 + 排版 + 色温
- **茑屋书店** —— 书架间的留白节奏
- **Notion 官网（旧版）** —— 米白底 + 绳结小图标
- **Linear.app** —— 现代克制
- **大谷彩子的禅画** —— 一笔一墨的呼吸感
- **京都町家室内设计** —— 木与纸的关系
- **加纳治五郎《柔道大要》初版书页**

### 关键情绪锚点（不是 spec · 是 feeling）

- 「克制」是核心 —— 少即是多，留白比内容重要
- 「木与纸」的物质暗示 —— 不要冰冷的玻璃感
- 「呼吸」—— 元素之间要有空气
- 「不刻意」—— 不要追求"日式符号"（樱花 / 富士山 / 毛笔字招牌全部 ❌）

### 颜色 · 给方向不给 HEX

往 **米白 + 木色 + 淡灰**走，accent 可以是 **黄绿（嫩茶色）** 或 **麦色** 或 **淡赭石**。整体饱和度低，温度偏暖。**避免**纯白（太刺眼）、纯黑（太重）、任何明亮原色。

### 字体 · 给方向不给字体名

主视觉用衬线（思源宋体 Light / Noto Serif JP / Cormorant），让标题有呼吸；正文用人文 sans（思源黑体 / Noto Sans JP / Avenir）；数字可以是 sans 也可以是 serif，**自己选哪个更呼应 mood**。

### 卡片 / 容器 / 装饰

- 卡片像翻开的素描本一页 —— 有纸感但不重
- 圆角 mild（不锐利不圆润，介于）
- 投影极轻（甚至可以没有，只用 hairline）
- **不要**毛玻璃 / 渐变 / 任何"科技感"装饰

### Lifespan 卡 · 必须出 2-3 个变体（重点）

这张卡是产品门面，决定高级感。请基于本 mood 出以下方向：

- **变体 A · 保守版**：维持上下两半结构，但用米白纸感 + 衬线主数字 + 等宽副文案。中间用极淡的木纹色 hairline 分隔。
- **变体 B · 创新版**：像翻开的素描本一页 —— 主数字"2,288"用细衬线像铅笔写下，副文案像页边批注，纸边轻微 noise 纹理。
- **变体 C · 大胆版（可选）**：极致留白 —— 整卡 60-70% 是空白，「2,288 周」浮在卡的右下角小字处，左上角只有一条极细横线和"时间余额"小字。整卡像一首俳句。

把 2-3 个变体在 HTML 里**竖向并列展示**，每个上方加 label "变体 A / B / C"。高度 ≤ 240pt 但允许超到 320pt 如果设计上更出彩。

### 整体感

像一本米色道林纸装订的手账。每翻一页都让你想慢一下。**不是日式刻板，是日式生活方式的当代提炼**。

## 5. 硬约束（只锁这 3 条 · 其余自由发挥）

1. **数据真实**：必须用 §3 林晚的精确数字，不要自己编占位（"100 hours"、"5 moments"等都不行）
2. **双数字双层结构**：lifespan 卡的"时间余额 + 已存入"两段都要 / 6 张时间账户卡的消耗层 + 已存入两数字都要，这是产品识别物
3. **文案逐字**：不要改"陪父母"为"父母"、不要改"已存入"为"已记录"、不要改"还能共度"为"剩余"

剩下：颜色、字体、间距、装饰、结构、字号反差、动效（如果 HTML 加）—— **全部你自己定**。
